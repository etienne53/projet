Diocèse de Laval – Plateforme web sécurisée de ressources liturgiques

PHP MySQL XAMPP JavaScript Gestion des droits

Contexte

Le Diocèse de Laval a besoin d’une plateforme numérique centralisée pour mettre à disposition des paroisses et des agents pastoraux des ressources liturgiques : missels, lectionnaires, partitions de chants, textes officiels, etc. L’objectif est de sécuriser l’accès, faciliter la recherche documentaire et assurer une gestion fine des droits (consultation, téléchargement, modification).

Fonctionnalités principales

  • Recherche documentaire avancée – par mots-clés, type de document, date, auteur.
  • Gestion des utilisateurs et des droits (administrateur, modérateur, simple lecteur).
  • Upload sécurisé de fichiers (PDF, images, audio) avec validation des métadonnées.
  • Catégorisation et étiquetage (liturgie des heures, messes, baptêmes, mariages…).
  • Statistiques de consultation pour les administrateurs.
  • Responsive design – accessible sur tablette et smartphone.

Architecture technique

Le projet a été développé avec l’environnement XAMPP (Apache, MySQL, PHP, Perl).

  • Back-end : PHP procédural et orienté objet pour la gestion des sessions, des droits et des requêtes SQL.
  • Base de données : MySQL avec des tables pour les utilisateurs, les rôles, les documents, les catégories et les logs.
  • Front-end : HTML5, CSS3, JavaScript (fetch API pour les recherches asynchrones).
  • Sécurité : protection contre les injections SQL (requêtes préparées), hachage des mots de passe (password_hash), validation des fichiers uploadés, HTTPS en production.
